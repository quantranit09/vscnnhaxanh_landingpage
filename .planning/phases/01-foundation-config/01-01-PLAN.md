### Task 2: Configure next.config.js for static export

<read_first>
- `next.config.js` or `next.config.ts` — replace entirely with static export config
</read_first>

<action>
Delete any existing `next.config.ts` or `next.config.mjs` and create `next.config.js` with this exact content:

```js
/** @type {import('next').NextConfig} */
const nextConfig = {
  output: 'export',
  trailingSlash: true,
  images: {
    unoptimized: true,
  },
}

export default nextConfig
```

NOTE: We set `unoptimized: true` as a safe base. We will wrap with `next-export-optimize-images` in Task 4.
</action>

### Task 4: Wrap next.config.js with next-export-optimize-images

<read_first>
- `next.config.js` — add withExportImages wrapper
</read_first>

<action>
Replace `next.config.js` with this final content:

```js
import withExportImages from 'next-export-optimize-images'

/** @type {import('next').NextConfig} */
const nextConfig = {
  output: 'export',
  trailingSlash: true,
}

export default withExportImages(nextConfig)
```

Note: Remove `images: { unoptimized: true }` — `withExportImages` handles this automatically.
</action>
