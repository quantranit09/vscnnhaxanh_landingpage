# Plan 01-03: Layout Shell + SEO Infrastructure

## Objective
Build the persistent layout shell (Header, Footer, Floating CTA) that appears on every page, implement SEO infrastructure (title/meta, JSON-LD LocalBusiness schema, sitemap, robots.txt), and create the Privacy Policy page required for Google Ads compliance.

## Placeholder Values (to be replaced by user)
- Hotline: `0905-000-000` → user replaces with real number
- Zalo: `https://zalo.me/0905000000` → user replaces
- Address: `Đà Nẵng, Việt Nam` → user provides full street address
- Domain: `https://nhaxanh-danang.vn` → user confirms domain
