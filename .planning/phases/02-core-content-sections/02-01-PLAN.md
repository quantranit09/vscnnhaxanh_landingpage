# Plan 02-01: Hero Section

## Objective
Build the above-the-fold Hero section — H1 với primary keyword, tagline thương hiệu, dual CTA (gọi ngay + báo giá), social proof badge "100+ khách hàng", và visual background gradient xanh. Đây là section đầu tiên người dùng thấy — phải convert ngay.

## Design Decisions (opinionated)
- **Layout:** Split — text bên trái (60%), visual element bên phải (40%) trên desktop; stack trên mobile
- **Background:** Gradient xanh từ #2D8B3A → #1E6B2A ở bên phải panel; trắng ở text panel
- **Alternative:** Full-width gradient overlay với ảnh nền (nếu user có ảnh)
- **CTA Primary:** "Gọi Ngay — 0905-000-000" (tel: link, màu xanh đậm)
- **CTA Secondary:** "Nhận Báo Giá Miễn Phí" (scroll to form #lien-he, outline)
- **Animation:** Framer Motion `motion.div` fadeInUp cho text, stagger children
