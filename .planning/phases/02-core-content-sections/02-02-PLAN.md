# Plan 02-02: Services Section

## Objective
Build the dịch vụ section với 5 service cards, section ID `#dich-vu` để anchor nav từ Header hoạt động. Mỗi card có icon Lucide, tên dịch vụ, mô tả 2-3 câu thực tế, và CTA "Báo giá dịch vụ này". Responsive: 1 cột (mobile) → 2 → 3 (desktop).
