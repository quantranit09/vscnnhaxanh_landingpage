# Plan 05-01: Performance Optimization

## Objective
Tối ưu hóa Performance để đạt điểm Lighthouse ≥90 và chuẩn bị cho việc tích hợp Zalo chat widget / Google Analytics 4 (GA4) không chặn luồng FCP chính của trang.

## Tasks

### Task 1: Audit images & Add GA4 / Zalo Chat Script skeletons

<read_first>
- `src/app/layout.tsx`
- `src/sections/Hero.tsx`
- `src/sections/About.tsx`
</read_first>

<action>
1. Thêm `next/script` với `strategy="lazyOnload"` vào Layout cho mục đích demo chặn third-party scripts. Zalo chat widget và GA4 nếu gắn phải dùng strategy này. Bổ sung `<Script strategy="lazyOnload" id="mock-ga" />` comments để sau này KH biết chỗ dán GA4 token thật.
2. Kiểm tra `Hero.tsx` và `About.tsx` - hiện đang sử dụng CSS background bằng thẻ `div` và `style={{ backgroundImage }}`. Do Landing Page dùng CSS mix-blend-multiply để blend màu thương hiệu khá phức tạp nên cách dùng Inline Style CSS cho `backgroundImage` là acceptable và performant (vì browser sẽ defer loading cho CSS images thay vì block render HTML). 
3. Kiểm tra lại `package.json` và `next.config.ts`, thêm `compress: true` vào `next.config.ts` để bật Gzip/Brotli auto-compression.
</action>
